Web Browser:

Web Broser
A web browser is a software application for accessing information on the World Wide Web.
  • Internet Explorer
    Internet Explorer is a series of graphical web browsers developed by Microsoft and included in the Microsoft Windows line of operating systems.

  • Google chrome
    Google Chrome is a cross-platform web browser developed by Google.
    It was first released in 2008 for Microsoft Windows, and was later ported to Linux, macOS, iOS, and Android where it is the default browser built into the OS.

  • Mozila firefox
    A free, open source, cross-platform, graphical web browser developed by the Mozilla Corporation and hundreds of volunteers.
    Firefox uses the Gecko layout engine to render web pages, which implements current and anticipated web standards.

  • Safari
    Safari is a graphical web browser developed by Apple, based on the WebKit engine.

  • Opera
    Opera is a multi-platform web browser developed by Opera Software. Opera is a Chromium-based browser.
